Projects

Anonymous Chat Platform

A real-time anonymous messaging platform built using React, Node.js, Express, and Socket.IO. The application enables users to communicate instantly without creating an account through WebSocket-based communication, ensuring fast and reliable message delivery. The backend is designed with scalable architecture, secure session management, REST APIs, and database integration for efficient performance.

Autonomous Obstacle-Avoiding Robotic Car

An autonomous robotic vehicle developed using ESP32, ultrasonic sensors, and motor drivers. The project detects obstacles in real time and automatically changes its path to prevent collisions. Sensor response and movement logic were optimized to provide stable navigation while demonstrating embedded systems concepts and seamless hardware-software integration.

KIIT Student Hub

A centralized web platform designed to help students access academic resources, placement preparation materials, and campus event updates from a single portal. The application includes secure user authentication, role-based access control, content management features, scalable REST APIs, and optimized database operations for improved performance and usability.
